## Introduction

Link Bird is an open-source API-first URL shortener

## Features

- **Analytics**
- **Geo-Targeting**
- **Custom Open Graph**
- **Webhook**
- **Expiry**

## Get Started

```bash
git clone https://github.com/hirebarend/lnkbrd.git

cd lnkbrd

npm install

npm run dev
```
